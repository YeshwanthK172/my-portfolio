## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/YeshwanthK172/my-portfolio.git
cd my-portfolio
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```

## 📝 Customization

## 📸 Adding Profile Photo

1. Add your image to:

```text
src/assets/images/
```

2. Import it into the greeting component:

```jsx
import profilePhoto from "../../assets/images/yeshwanth.png";
```

3. Display it using an `<img>` tag and apply styling.
